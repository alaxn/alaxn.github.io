安装Hexo

1. 在想要安装的文件夹空白处按住Shift+鼠标右键，打开命令行窗口（Win10需注意权限，权限不足需使用管理员模式运行）
2. 输入：
`npm install hexo-cli -g`
完成后弹出WARRING不要紧
3. 接着输入：
`npm install hexo --save`
4. 检查是否安装成功：
`hexo -v`
如果返回安装版本信息，则安装成功。

初始化Hexo

1. 输入：
`hexo init`
2. 输入：
`npm install`
3. 安装完成
