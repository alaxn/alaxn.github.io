Hexo 部署中的文件大小写问题

Posted on 2017-07-03 | In MachineWorld , Hexo | Visitors

问题描述

修改文件或文件夹名称后无法成功部署，例如原来的文章名称是Hexo.md，后来改为hexo.md，但上传到github并不会生效新的名称。

原因探讨

git默认忽略文件大小写。

解决办法

方法一

如果只是一两个文件，可以直接删除文件，再进行发布。

注意，此方法不能一劳永逸

方法二

  1. 进入到博客项目中 .deploy_git文件夹，修改 .git 下的 config 文件，将 ignorecase=true 改为 ignorecase=false。
  2. 删除博客项目中 .deploy_git 文件夹下的所有文件，并 push 到 Github 上。
  3. 再次生成发布。

个人推荐的Win应用

Posted on 2017-06-19 | In MachineWorld | Visitors

浏览器

浏览器看个人喜好，但建议不使用国产浏览器，尤其是360和猎豹，原因是隐私安全

  • Mozilla Firefox：安全性高，速度快，拓展性好
  • Chrome：理由同上
  • Edge：微软新浏览器，速度挺快
  • Google Chromium：Beta版的Chrome，有时有些新功能，不过偶尔不稳定

视频播放器

个人只是一般视频爱好者，一般使用，勿喷：）

  • Daum Potplayer：个人最喜爱的播放器，简洁无广告，可播放大多数格式
  • VLC media player：开源软件，自带功能较少，需调教
  • KMP Player：和字符串匹配算法KMP同名，也挺喜欢的，但后来似乎越更新越差了？
  • QQ影音：轻便，也支持主流格式，已经很久未更新了

音乐播放器

  • Foobar2000
  • 酷狗音乐绿色版

PDF阅读器

  • 福昕阅读器Foxit Reader
  • Adobe Acrobat Pro：主打OCR识别
  • ABBYY FineReader：同样主打OCR识别

下载类

  • IDM：多线程下载软件，不支持bt，配合猴油可满载or高速下载百度云资源
  • Free Download Manager
  • 迅雷精简版
  • 硕鼠

格式转换

  • FormatFactory：转换神器之一，绿色版即可
  • Caesium

编辑器

  • Notepad++
  • Sublime Text
  • typora：个人最爱的Markdown编辑器

辅助软件

  • f.lux：f.lux能随着时间变化，自动调整屏幕的色温，减少屏幕对眼睛所带来的疲劳感。注Win10有自带功能
  • Unlocker：可解除文件锁定或占用情况
  • Ditto：保存剪辑版，方便多次重复使用某些内容
  • Everthing：搜索神器
  • PrimoCache：利用内存及内存卡作为高速缓存，提高硬盘读写速度，减少SSD碎片化写入
